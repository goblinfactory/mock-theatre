# Mock Theatre Facilitator Notes (20-minute version)

This is the compressed version of the "Mock Theatre: when green tests mean nothing"
session for a 20-minute slot.

## How to use these notes

- Each topic to 5-6 minutes max.
- At most one 2-minute prompt.
- Demos to 30-45 seconds.
- One SME share at the end (if available).

## Quick links

- [01. Intro and framing](01-intro.md)
- [02. Testing pyramid and test types](02-testing-pyramid.md)
- [03. The useless test example](03-useless-test-example.md)

## Suggested flow (20 minutes)

| Time | Segment | Notes |
| --- | --- | --- |
| 0-3 | Intro and framing | Define mock theatre and why it matters |
| 3-9 | Testing pyramid and test types | Place each test type in context |
| 9-15 | The useless test example | Demonstrate the worst case |
| 15-18 | One 2-minute prompt | Pick the most relevant prompt |
| 18-20 | Wrap-up | Reference the 90-minute version |

## Materials checklist

- Shared whiteboard link prepared
- Timer for the prompt

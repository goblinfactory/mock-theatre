# Mock Theatre Facilitator Notes (90-minute version)

These notes are designed to run the unconference session "Mock Theatre: when green tests mean nothing" in a 90-minute slot.

## How to use these notes

- Each topic file is a self-contained facilitation page.
- Suggested flow: context -> quick demo -> 2-minute prompt -> share-out.
- SME response after each prompt (when available).
- Shared whiteboard open (Miro, Hackerdraw, or similar) for remote participants.

## Suggested flow (90 minutes)

| Time | Segment | Notes |
| --- | --- | --- |
| 0-8 | Intro and framing | Set the stakes and define "mock theatre" |
| 8-22 | Chicago vs London TDD | Align on styles and tradeoffs |
| 22-36 | Testing pyramid and test types | Place each test type in context |
| 36-50 | When tests lie | Failure modes and symptoms |
| 50-65 | Useless test example | Demonstrate the worst case |
| 65-80 | Facilitation playbook | One short collaborative exercise |
| 80-90 | Open share-out | SME stories and Q&A |

## Time-saving options

- Only one 2-minute prompt every other topic.
- Full facilitation playbook can be skipped; guidance applied inline.
- Demos kept to 60 seconds max.

## Materials checklist

- Shared whiteboard link prepared
- Timer for the 2-minute prompts
- Repo open at this index page
